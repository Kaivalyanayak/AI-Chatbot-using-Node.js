# Interactive AI ChatBot

A simple AI-powered chatbot web app built with Node.js and Express. 
It uses Groq's LLaMA 3.3 model to generate responses and MongoDB to store chat history.

## Features
- Real-time chat interface
- AI-generated responses via Groq API
- Chat history saved to MongoDB
- Send messages via button click or Enter key
- Markdown rendering for formatted bot responses

## Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- EJS templating
- Groq API (OpenAI-compatible)
